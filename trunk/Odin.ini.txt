Odin initialization file documentation

[COLORS]
UseWinColors	Use Windows colors
		values: 1/0
		default: 0
SaveColors	Save/restore colors to/from Odin.ini
		values: 1/0
		default: 0

[DIRECTORIES]
SYSTEM		Windows system directory
		- This directory _must_ exist!
		- Don't use backslashes as last character!
		default: Open32 -> \os2\system
		example: e:\win32\system
WINDOWS		Windows directory
		- This directory _must_ exist!
		- Don't use backslashes as last character!
		default: Open32 -> \os2
		example: e:\win32

[DLLRENAMEWIN]
OLE32=OLE32OS2
NETAPI32=WNETAP32

[DLLRENAMEOS2]
OLE32OS2=OLE32
WNETAP32=NETAPI32

[TWAIN]
TwainIF    	values 1,2,3 (default=1)
		1 -> Use CFM Twain Driver
		2 -> Use STI Twain Driver (not supported at the moment)
		3 -> Use SANE Driver (not supported at the moment)

[CODEPAGES]
Windows         Windows codepage (i.e. 1251 for Cyrillic)
Display         Display codepage (i.e. 1251 for Cyrillic)

Odin.ini is stored in the directory where the environment string
ODIN_INI points to. The default path is the directory of
KERNEL32.DLL.
